Développeur Full Stack

Tout d'abord, dans la programation web, il existe trois types de développeur :


- Le développeur Front-End : Il s'occupe de toute la partie visuelle du site web, la partie émergée de l'iceberg, il va souvent intégrer des maquettes faites par des UI/UX designer, son rôle est de rendre le site agréable à l'oeil et d'offrir la meilleure experience utilisateur.

- Le développeur Back-End : Il s'occupe de toute la partie immergée de l'iceberg, il s'occupe des bases de données, de la gestion d'API, son travail est nécessaire au bon fonctionnement du site web, cependant, celui-ci ne sera pas visible des utilisateurs contrairement au développeur Front-End.

- Le développeur Full Stack : Il est comme un couteau suisse, il sait tout faire, en passant par de l'intégration web, de la gestion de bases de données.

Les langages de programmation Full Stack

Etant donné qu'un développeur Full Stack se doit d'avoir des compétences en Front-End et Back-End, il doit acquérir beaucoup de connaissances.

Il doit donc maîtriser plusieurs languages de programmation :

Javascript

Javascript est un language qui est utilisé tout aussi bien dans le Front-End, que dans le Back-End
console.log("Hello World")

Python

Python est le langage le plus utilisé dans le monde, il offre de nombreuses possibilitées, iol s'occupe plus du côté Back-End
print("Hello World")

PHP

PHP est un langage impératif orienté objet, le php est un langage principalement conçu pour crér des pages web dynamiques
<?php echo 'Hello World !'; ?>

Ruby

Ruby est un langage essentiellement Back-End orienté objet
"Hello World"

React

React est une librairie javascript qui facilite la création d'applications web en single page
ReactDOM.render( <?h1>Bonjour, monde !<?/h1>, document.getElementById('root') );

SQL

SQL est un language Back-End qui sert à gérer des bases de données
select 'Hello, world';
